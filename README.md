# 坐姿檢測系統(此部分僅為分類器部分)

## 主要使用Google的Posenet模型
搭配網路攝影機使用，獲取使用者的人體座標點，使用分類器判斷使用者坐姿是否正確，將結果回傳給Firebase資料庫。APP的部分自FIrebase上擷取結果，使用者維持正確坐姿特定時間長短，可獲得角色故事，否則APP會發出警示音。
